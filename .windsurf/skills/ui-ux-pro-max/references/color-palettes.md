# Color Palettes Reference

## SaaS Dashboard Palettes

### Trust Blue (MEGVAX Default)
- Primary: `#2563EB` (blue-600)
- Hover: `#1D4ED8` (blue-700)
- Light: `#DBEAFE` (blue-100)
- Accent: `#3B82F6` (blue-500)
- Background: `#F9FAFB` (gray-50)
- Surface: `#FFFFFF`

### Functional Colors
- Success: `#16A34A` (green-600) — positive metrics, active states
- Warning: `#D97706` (amber-600) — waiting, attention needed
- Danger: `#DC2626` (red-600) — errors, destructive actions
- Info: `#2563EB` (blue-600) — informational badges

### Neutral Scale
- 900: `#111827` — primary text
- 700: `#374151` — secondary text
- 600: `#4B5563` — descriptions
- 500: `#6B7280` — muted text, labels
- 400: `#9CA3AF` — placeholders
- 200: `#E5E7EB` — borders, dividers
- 100: `#F3F4F6` — subtle backgrounds
- 50: `#F9FAFB` — page background

## Dark/Premium Palettes (Marketing)

### Midnight Gradient
- Base: `#0A0A0F`
- Surface: `#111827`
- Accent 1: `#3B82F6` → `#8B5CF6` (blue to purple gradient)
- Accent 2: `#06B6D4` → `#3B82F6` (cyan to blue gradient)
- Text: `#F9FAFB`

### Neon Accent
- Base: `#030712`
- Glow: `#22D3EE` (cyan-400)
- Accent: `#A78BFA` (violet-400)
- CTA: `#F59E0B` (amber-500)

## Palette Selection Guide

| Context | Recommended | Avoid |
|---------|-------------|-------|
| Dashboard data | Trust Blue + Functional | Bright/saturated backgrounds |
| Marketing hero | Dark gradients | Plain white backgrounds |
| Forms/inputs | Neutral + blue focus | Red focus rings (implies error) |
| Charts | Blue → Green → Amber → Red | More than 5 colors in one chart |
| Status badges | Semantic (green/amber/red) | Arbitrary colors |
