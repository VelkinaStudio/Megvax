# MegVax Redesign -- Design Reference Sites

**Date:** 2026-03-25
**Purpose:** Curated reference sites for redesigning MegVax (Meta Ads SaaS platform)
**Tier:** Deep research (8 searches, 10+ site fetches)

---

## The List (12 Sites)

### 1. Linear -- https://linear.app
**Category:** Landing page + animations + overall brand
**What makes it exceptional:**
- Dark theme with precisely calibrated text hierarchy (primary, secondary, tertiary, quaternary colors)
- Staggered dot-grid animations (3200ms cycles) that pulse without overwhelming
- Monospace font for technical elements paired with a clean primary typeface
- Every pixel of spacing, letter-spacing, and font-feature-settings is intentional

**Steal for MegVax:**
The multi-layered text color system. MegVax shows a lot of data (ROAS, spend, CTR). Linear proves you can have dense information that feels calm by using 4 tiers of text opacity instead of bold/color for hierarchy.

---

### 2. Stripe -- https://stripe.com
**Category:** Landing page + animations + overall brand
**What makes it exceptional:**
- Continuously flowing gradient animations (blues, purples, yellows, pinks, oranges) -- the most famous homepage animation in SaaS
- Card-based bento grid layout for product categories
- "$1.9T processed" stat placement that builds instant authority
- Clean grayscale palette with strategic accent colors for trust + sophistication

**Steal for MegVax:**
The bento-grid product showcase. MegVax has multiple modules (Creative AI, Budget Optimizer, Analytics). Present them in Stripe-style cards with embedded product screenshots, not generic icons. Also: a single bold metric in the hero ("$X managed ad spend") for credibility.

---

### 3. Vercel -- https://vercel.com
**Category:** Landing page + performance storytelling
**What makes it exceptional:**
- Dark-first, light-adaptable palette with system `prefers-color-scheme` detection
- Custom Geist fonts (sans + mono) that became an industry standard
- Performance metrics as hero content ("7m to 40s builds", "95% faster page loads")
- Globe with real-time node pulses for edge network visualization
- Container queries for fluid responsive design

**Steal for MegVax:**
The "performance metrics as hero" approach. Instead of "Optimize your Meta Ads," show: "Average 2.4x ROAS improvement in 30 days" with real numbers. Also steal the globe/network visualization concept -- adapt it to show Meta's ad delivery network or audience reach.

---

### 4. Raycast -- https://raycast.com
**Category:** Landing page + animations + dark mode reference
**What makes it exceptional:**
- Deep black/navy backgrounds with vibrant magenta, teal, and electric blue accents
- Animated 3D cube with glass-effect surfaces that responds to cursor
- Glassmorphism (semi-transparent overlays with backdrop blur) adding depth
- Monochromatic base with selective bold red for CTAs
- Progressive vertical scroll revealing features one by one

**Steal for MegVax:**
The glassmorphism + dark mode aesthetic. MegVax users (media buyers) work late, stare at dashboards. A dark-first landing page with glassmorphic card overlays showing ad performance data would feel native to their workflow. The vibrant accent colors on dark backgrounds make data pop.

---

### 5. Huly -- https://huly.io
**Category:** Animations + dark mode + motion storytelling
**What makes it exceptional:**
- Hero video background with `clip-path: circle()` dynamic masking
- Gradient text effect (white -> #d5d8f6 -> #fdf7fe) creating ethereal purple-pink glow
- Infinite marquee scroll for feature highlights
- `blur-3xl` radial gradients with `mix-blend-lighten` for depth
- Dark-to-light section transitions across the page
- 80-84px title font with snug tracking

**Steal for MegVax:**
The motion-based storytelling. Instead of static sections, MegVax should animate the "how it works" flow: show an ad being created -> optimized by AI -> performance improving -- all through scroll-triggered video/animation sequences. Huly proves you can do heavy motion without feeling gimmicky.

---

### 6. Clay -- https://clay.com
**Category:** Landing page + data platform presentation
**What makes it exceptional:**
- Space Mono typeface with custom OpenType features (ss01, ss03) for distinctive character
- Named color tokens (tangerine, blueberry, matcha, pomegranate) instead of generic hex
- Staggered animations with `data-stagger-list` guiding attention progressively
- Quantified testimonials ("3x enrichment rate", "3M+ companies monitored")
- "Clayland" footer illustration humanizing an enterprise product
- Pausable integration marquees

**Steal for MegVax:**
The quantified social proof approach. Instead of "Trusted by 500+ brands," show specific results: "Average 47% reduction in CPA" or "3.2x ROAS for DTC brands." Clay proves data platforms should lead with outcomes, not features. Also: the named color system is excellent for a design system.

---

### 7. Mercury -- https://mercury.com
**Category:** Overall brand + landing page + fintech-grade trust
**What makes it exceptional:**
- Purple-themed palette with complementary neutrals -- sophisticated without being cold
- Smartphone mockup in hero showing actual dashboard (not generic mockups)
- Smooth `opacity + translateY` entrance animations on scroll
- Consistent spacing system using design tokens (s8, s9, s12, s20)
- Testimonial sections that humanize the product
- `outline-offset-4` focus states for accessibility polish

**Steal for MegVax:**
The fintech-grade trust design. MegVax handles ad budgets (real money). Mercury's approach -- purple tones, real dashboard screenshots, precise spacing tokens, human testimonials -- builds the "safe to trust with my money" feeling that MegVax needs. Show the actual MegVax dashboard in the hero, not illustrations.

---

### 8. Madgicx -- https://madgicx.com
**Category:** Direct competitor reference (Meta ads platform)
**What makes it exceptional:**
- Dark theme with purple (#515FBC) accent -- the closest competitor aesthetic
- Rotating gradient borders (CSS spin animation, 3s cycle)
- Dual-angle conic gradient glowing effects on cards
- Modular sections with animated dividers between content blocks
- Dashboard screenshots embedded throughout to show product capability
- "Dominate AI Ads Results" aggressive positioning

**Steal for MegVax (and improve upon):**
Study Madgicx as the benchmark to surpass. Their purple-dark theme works for the ad-tech audience but their execution is mid-tier. MegVax should take the dark+purple direction but execute at Linear/Stripe quality. Better typography, smoother animations, more whitespace, less aggressive copy. Beat them on craft.

---

### 9. Loops -- https://loops.so
**Category:** Landing page + audience segmentation + developer-friendly design
**What makes it exceptional:**
- Role-based content sections (Designers, Developers, Marketers, Everyone)
- Code examples and API docs integrated naturally into marketing pages
- "Simplicity meets polish" as both tagline and design philosophy
- Customer logos (Framer, Spline) as social proof without clutter
- Confident whitespace -- not afraid of empty space

**Steal for MegVax:**
The role-based landing experience. MegVax serves different users: media buyers, creative teams, agency owners, brand managers. Instead of one generic page, show different value props per role -- like Loops does for designers vs developers vs marketers. Each persona sees their pain point addressed first.

---

### 10. Framer -- https://framer.com
**Category:** Overall brand + marketplace showcase
**What makes it exceptional:**
- Dark mode with minimalist palette using weight variation (not color) for hierarchy
- Full-width hero showing actual product output (a real website built in Framer)
- Feature-focused layout where AI, Design, CMS, Collaborate each get visual real estate
- Marketplace carousel demonstrating ecosystem richness
- "Premium through restraint" -- no ornamentation, just quality imagery and motion

**Steal for MegVax:**
The "show, don't tell" hero. Framer's hero IS a website built with Framer. MegVax's hero should BE a dashboard showing real ad performance data (anonymized). Let the product demonstrate itself. Also: the feature categorization (AI, Design, CMS) maps well to MegVax modules (Creative AI, Budget AI, Analytics, Audiences).

---

### 11. Attio -- https://attio.com
**Category:** Dashboard + overall brand + data-rich SaaS
**What makes it exceptional:**
- Sleek monochromatic layout with bold typography
- Dynamic light-to-dark transitions across page sections
- Apple-product-level interface intuitiveness
- Data visualization through thoughtful spacing and typography (not chart overload)
- "Ask more from CRM" -- confident, short positioning

**Steal for MegVax:**
The light-to-dark section transitions. Instead of a static dark page, alternate between light sections (social proof, testimonials) and dark sections (product demos, dashboards). This creates visual rhythm and prevents dark-mode fatigue. Also: Attio's confident 4-word tagline approach. MegVax needs something equally sharp.

---

### 12. Notion -- https://notion.com
**Category:** Landing page + illustration style + approachability
**What makes it exceptional:**
- Multi-color section blocking (teal, red, blue, yellow) creating energy and separation
- Playful animations ("noseyAnimation" elements) adding personality
- Video replacing static images in key sections
- "Now a team of 7 feels like 70" -- conversational, human copy
- Dual CTA buttons in hero for different commitment levels
- Balances enterprise credibility with startup approachability

**Steal for MegVax:**
The color-blocked sections for visual energy. A fully dark landing page can feel monotonous. Notion proves you can inject color blocks for specific sections (pricing, testimonials, case studies) while maintaining sophistication. Also: the conversational copy style works well for making complex ad-tech feel accessible.

## Top Design Principles for MegVax (Synthesized)

1. **Dark-first, data-native.** Media buyers live in dark dashboards. Meet them there.
2. **Metrics as hero.** Lead with outcomes (ROAS, CPA reduction), not features.
3. **Show the product.** Real dashboard screenshots > illustrations > stock photos.
4. **4-tier text hierarchy.** Dense data needs opacity layers, not bold/color chaos.
5. **Motion with purpose.** Scroll-triggered storytelling, not decorative parallax.
6. **Role-based messaging.** Media buyer / agency owner / brand manager see different things.
7. **Trust through craft.** Precise spacing, custom fonts, smooth animations = "safe to spend budget here."
8. **Beat Madgicx on execution.** Same dark+purple direction, Linear-level quality.
